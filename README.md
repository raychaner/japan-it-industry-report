# 日本IT行业现状和趋势调查报告

本仓库是基于 Docsify 的静态报告站点，所有内容以 `docs/` 作为唯一信息源。

## 在线阅读

https://raychaner.github.io/japan-it-industry-report/

## 内容与编辑

- 站点首页与总览在 `docs/README.md`
- 章节与图表在 `docs/` 下的各章文件
- 项目大纲在 `docs/00_项目大纲.md`

## 本地运行

```bash
npm i docsify-cli -g
docsify serve docs
```

或使用 Python：

```bash
cd docs
python -m http.server 3000
```
