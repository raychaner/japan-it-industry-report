# 日本IT行业现状和趋势调查报告

本项目是一份面向日本 IT 行业的调查报告，覆盖市场规模、人才现状、雇佣结构、外国人就业、行业机会、数字化转型与未来展望等主题。报告正文以 `docs/` 为唯一信息源，本 README 提供摘要与导航。

## 在线阅读

https://raychaner.github.io/japan-it-industry-report/

## 适合读者

- 想来日本工作的外国 IT 人才
- 在日本的 IT 从业者
- 日本 IT 企业管理者
- 投资者/分析师
- 学术研究者
